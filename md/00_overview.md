内容简介

本书对Spark应用程序开发的基本概念和技术进行了系统的介绍，并通过简单易懂的实例说明了其具体实现过程。通过本书的学习，读者可以牢固掌握Spark编程技术的基本概念、原理和编程方法，通过灵活的实践运用，能够进行应用程序的实际开发。

本书适用于Spark程序设计的初学者，适合作为高等学校计算机专业的教材，还可以作为Spark程序设计的培训教材。

## 怎么读这本书（通俗版）
- 先看每章开头的“最小例子”，把概念和代码先建立直觉。
- 再按“概念 -> API -> 实战”的顺序阅读，不必一次看完所有细节。
- 学习时建议边看边在本地 `spark-shell` 或 `pyspark` 里复现。

## 一个最小例子
假设你有一份订单数据，想快速算出“每个城市的订单数”：
1. 读入数据（CSV/Parquet）。
2. 按城市分组。
3. 做 `count` 聚合。
4. 输出结果并观察是否符合业务直觉。

这就是本书很多示例的基本模式：读数据、变换、聚合、输出。
